```
let x = 10;
const s = make_serializer();
concurrent_execute(
    () => { x = s(() => x * x)(); },
    s(() => { x = x + 1;})
);
```

> 101, 121, 110, 11, 100

- 사라지는 것: 11, 110
- 남는 것: 121, 100, 101

---

- 121

1. s(() => { x = x + 1;}) // 11
2. s(() => x \* x)(); // 11 (할당 전, 계산값 121)
3. x = s(() => x \* x)(); // 121

- 100

1. s(() => x \* x)(); // 10 (할당 전, 계산값 100)
2. s(() => { x = x + 1;}) // 101
3. x = s(() => x \* x)(); // 100

- 101

1. s(() => x \* x)(); // 100
2. x = s(() => x \* x)(); // 100
3. s(() => { x = x + 1;}) // 101

- 11

1. s(() => x \* x)(); // 100
2. s(() => { x = x + 1;}) // 할당하기 전 까지만 수행 (11, x는 10)
3. x = s(() => x \* x)(); // 100
4. s(() => { x = x + 1;}) // 11, 할당 수행
